# Api Manager

## Pré-requis

- Si ce n'est pas déjà le cas, [installer Volta](https://volta.sh/),
- Ou la version correspondante à Node (voir `package.json`).

## Development

```bash
cp .env.example .env
```

### Installation des dépendances

```
npm install
```

### Lancement en local

```bash
npm run dev
```

### Lint

```bash
npm run lint
```
